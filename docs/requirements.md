# 需求文档

> 第十五届中国软件杯大赛 · A组赛题
> 赛题名称：基于大模型的个性化资源生成与学习多智能体系统开发
> 出题企业：科大讯飞股份有限公司

## 一、赛题简介

在数字化与智能化深度融合的时代，高等教育的个性化变革成为核心发展方向，面临传统教学模式适配性不足的挑战。不同学生在知识基础、学习兴趣上存在显著差异，标准教学难以满足所有人需求。

随着通用多模态生成（SeeDance 等）、AI 辅助编程工具（Claude Code 等）为代表的技术体系成熟，本赛题旨在借助前沿 AI，突破局限，构建基于大模型的个性化教育系统。

## 二、业务场景

学习过程普遍繁杂无序，缺乏个性化指导：
- 对海量课程和学术文献，难以快速筛选出契合度和难度匹配的内容
- 课堂集体授课无法兼顾每位学生的节奏与特点
- 需要"因材施教"的可行落地

## 三、基本功能

### 1. 对话式画像
- 通过对话交互构建学习者画像
- 摒弃繁琐表单
- 至少 6 个维度：知识水平、认知风格、易错点、偏好、学习速度、理解深度

### 2. 个性化内容生成
- 至少 5 种类型：讲解、思维导图、题目、扩展阅读、视频/动画、代码

### 3. 学习路径规划
- 智能规划最优路径
- 动态调整策略

### 4. 智能答疑
- RAG 多轮对话
- 详细解答、逐步讲解

### 5. 评估与反馈
- 自动评估学习效果
- 跟踪进度
- 自适应调整策略

## 四、技术要求

- 流式输出、Markdown 渲染、卡片展示
- 防幻觉、安全过滤、无敏感违规内容
- 响应时间控制在合理范围内
- 支持 Web / 程序等多平台
- 统一格式规范
